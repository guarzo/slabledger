## 1. Campaign Detail stat card layout

Files: `web/src/react/pages/campaign-detail/OverviewTab.tsx`, `web/src/react/ui/StatCard.tsx`

1. **`lg` StatCard needs `col-span-2`.** The `size="lg"` variant sets typography but
   doesn't widen the card. Add `col-span-2` (or `col-span-full` at mobile) to the `lg`
   variant's className so the two primary cards (Total Spent, Net Profit) actually span
   two grid columns in the `md:grid-cols-2` layout.

2. **Hide Row 3 in zero state.** When `noSalesYet` is true, the `sm` inline stats
   (Sold / Unsold / Avg Days / Daily Cap) should also be hidden or collapsed into the
   EmptyState card. Currently only Row 2 collapses.
