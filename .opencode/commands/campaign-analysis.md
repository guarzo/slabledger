Use the `skill` tool to load the `campaign-analysis` skill, then execute its full workflow.

The argument passed to this command is: $ARGUMENTS

Pass that argument to the skill as the analysis mode (empty means Full Portfolio Overview, `health` for quick health check, `weekly` for weekly review, `tuning` for campaign tuning, or `campaign N` for a single campaign deep dive).
